# Photo-Me International - Next.js Website

A premium multi-page photography and videography website for **Photo-Me International**.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Pages
- Home
- About
- Services
- Portfolio
- Pricing
- Team
- Contact

## Run locally
```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Notes
- Placeholder images are used and can be replaced with actual brand images.
- Instagram CTA points to `@photome.ng`.
- Contact details are placeholder values and should be updated before launch.
